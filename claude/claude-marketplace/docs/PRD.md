# 맥주 큐레이션 서비스 PRD (Product Requirements Document)

## 1. 서비스 개요

### 1.1 서비스명
**BrewPick** — 나만의 맥주 큐레이터

### 1.2 서비스 한 줄 설명
사용자의 취향, 상황, 페어링 음식을 기반으로 최적의 맥주를 추천하고, 국내외 크래프트 맥주 탐색 경험을 제공하는 큐레이션 플랫폼

### 1.3 배경 및 문제 정의
- 국내 크래프트 맥주 시장은 연평균 15% 이상 성장 중이나, 소비자는 수백 종의 맥주 중 자신에게 맞는 제품을 선택하기 어려움
- 맥주 라벨이나 스펙만으로 맛과 어울리는 상황을 판단하기 힘들어 구매 실패 경험 반복
- 기존 주류 커머스는 단순 나열 방식으로 개인화 추천 기능 부재

### 1.4 목표
- 사용자 취향 기반 개인화 맥주 추천으로 탐색 시간 단축
- 맥주 입문자부터 마니아까지 모두 만족하는 정보 깊이 제공
- 커뮤니티와 리뷰 기반 신뢰도 높은 큐레이션 생태계 구축

---

## 2. 타겟 사용자

### 2.1 주요 타겟 (Primary)
| 페르소나 | 특성 |
|---|---|
| **맥주 입문자** | 20~30대, 다양한 맥주를 경험하고 싶지만 어디서 시작할지 모름 |
| **홈술족** | 집에서 분위기 있게 즐길 맥주를 편리하게 탐색·구매하고 싶음 |
| **선물 구매자** | 맥주를 좋아하는 지인에게 특별한 선물 세트를 고르고 싶음 |

### 2.2 보조 타겟 (Secondary)
- 크래프트 맥주 마니아 (상세 스펙, 희귀 제품 정보 탐색)
- 식당/바 운영자 (메뉴 구성을 위한 도매성 큐레이션)

---

## 3. 핵심 기능 (Features)

### 3.1 취향 온보딩

**목적**: 신규 사용자의 맥주 취향을 빠르게 파악하여 즉각적인 개인화 제공

**기능 상세**
- 5~7문항의 취향 설문 (선호 맛: 쓴맛/단맛/과일향/고소함, 도수 선호도, 평소 음주 빈도)
- 대표 맥주 3종 시음 기록 입력 (이미 마셔본 맥주 평가)
- 온보딩 완료 시 초기 프로필 생성 및 맞춤 추천 즉시 제공

**완료 기준**
- 온보딩 완료율 ≥ 70%
- 온보딩 소요 시간 ≤ 3분

---

### 3.2 AI 맥주 추천 엔진

**목적**: 상황·음식·기분을 고려한 맥락적 추천 제공

**기능 상세**
- **상황 기반 추천**: "혼자 영화 보며", "치킨이랑", "더운 여름날" 등 자연어 입력
- **페어링 추천**: 음식 선택 시 궁합 맥주 목록 제공 (예: 삼겹살 → 필스너, 페일에일)
- **기분/날씨 추천**: 날씨 API 연동으로 현재 날씨에 어울리는 맥주 자동 제안
- **유사 맥주 추천**: 마음에 든 맥주 기반 비슷한 스타일 탐색

**알고리즘 기반**
- 협업 필터링 (비슷한 취향의 사용자 리뷰 반영)
- 컨텐츠 기반 필터링 (맥주 스타일, IBU, ABV, 홉 특성 매칭)
- Claude API 활용 자연어 입력 해석 및 설명 생성

**완료 기준**
- 추천 클릭율(CTR) ≥ 35%
- 추천 만족도 평점 ≥ 4.0/5.0

---

### 3.3 맥주 상세 정보 페이지

**목적**: 구매 결정에 필요한 모든 정보를 한 곳에서 제공

**제공 정보**
- 기본 스펙: 스타일, 원산지, 양조장, 도수(ABV), 쓴맛 수치(IBU), 색상(SRM)
- 맛 프로필 레이더 차트 (쓴맛/단맛/신맛/과일향/몰트향)
- 추천 페어링 음식 목록
- 서빙 가이드 (적정 온도, 추천 글라스)
- 사용자 리뷰 및 평점
- 구매 가능 온라인몰 링크 (최저가 비교)

---

### 3.4 컬렉션 & 위시리스트

**목적**: 사용자가 탐색한 맥주를 체계적으로 관리하고 기록

**기능 상세**
- **마신 맥주 기록**: 날짜, 장소, 개인 평점, 노트 저장
- **위시리스트**: 마시고 싶은 맥주 저장
- **나만의 컬렉션**: 테마별 큐레이션 목록 생성 (예: "여름에 마시기 좋은 맥주")
- **연간 음주 리포트**: 마신 맥주 스타일, 원산지, 선호 패턴 시각화

---

### 3.5 커뮤니티 리뷰

**목적**: 신뢰 기반의 사용자 생성 콘텐츠로 추천 품질 향상

**기능 상세**
- 텍스트 리뷰 + 별점 (1~5점)
- 맛 태그 선택 (쓴맛강함/과일향/청량함 등 다중 선택)
- 리뷰 좋아요 / 유용함 투표
- 팔로우 기능: 취향이 비슷한 사용자 팔로우 및 리뷰 피드 구독

---

### 3.6 정기 구독 큐레이션 박스 (향후 확장)

**목적**: 오프라인 구매 연계로 수익 모델 다각화

**기능 상세**
- 월 1회 취향 맞춤 맥주 4~6종 큐레이션 박스 배송
- 동봉 카드: 각 맥주 소개 및 페어링 레시피
- 구독 플랜: 베이직(4종) / 프리미엄(6종+희귀 한정)

## 6. 사용자 스토리 (User Stories)

```
US-01: 맥주 입문자로서, 취향 설문을 완료하면 나에게 맞는 첫 맥주를 추천받고 싶다.
US-02: 사용자로서, "삼겹살에 어울리는 맥주"를 검색하면 페어링 추천 목록을 보고 싶다.
US-03: 사용자로서, 마음에 드는 맥주를 위시리스트에 저장하고 나중에 확인하고 싶다.
US-04: 사용자로서, 내가 마신 맥주를 기록하고 연말에 취향 리포트를 받고 싶다.
US-05: 사용자로서, 비슷한 취향의 다른 사용자 리뷰를 우선적으로 보고 싶다.
US-06: 운영자로서, 신규 맥주를 등록하고 스펙 정보를 관리하고 싶다.
```

## 8. 출시 로드맵

### Phase 1 — MVP (0~3개월)
- [ ] 취향 온보딩 설문
- [ ] 맥주 데이터베이스 구축 (국내 크래프트 500종)
- [ ] 기본 추천 엔진 (태그 기반)
- [ ] 맥주 상세 페이지
- [ ] 위시리스트 / 마신 맥주 기록

### Phase 2 — 성장 (4~6개월)
- [ ] AI 자연어 추천 (Claude API 연동)
- [ ] 커뮤니티 리뷰 시스템
- [ ] 페어링 음식 추천
- [ ] 날씨 연동 추천

### Phase 3 — 수익화 (7~12개월)
- [ ] 정기 구독 큐레이션 박스
- [ ] 온라인몰 최저가 비교 연동 (제휴)
- [ ] 프리미엄 멤버십 (광고 제거 + 독점 추천)
- [ ] 연간 음주 리포트 (Spotify Wrapped 형식)
